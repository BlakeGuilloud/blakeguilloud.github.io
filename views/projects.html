SchinDig

Schindig Demo Video

A mobile application that enhances the party planning experience, Schindig allows for hosts to select from a dynamically suggested list of party favors that become their party's registry.

As an invited guest to a party, one would have the option to claim a favor, RSVP, and pitch in towards a big ticket item, such as a band.

Schindig leverages phone hardware in order to manage invitations. Using Venmo's API, we also are able to auth a Venmo ID after the user has synced their account. Real money - movin' around.

  • AngularJS
  • Ionic
  • ngCordova
  • Venmo API
